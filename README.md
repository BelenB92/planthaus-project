#Plant Haus

##Descripción del proyecto:
Plant Haus es un ecommerce dedicado a la venta de plantas para el hogar.

##Instalación:
git clone 
npm install
npm run dev

##Tecnologías usadas:
-React Js
-Node
-Vite
-React Router Dom
-Firebase
-Bootstrap
